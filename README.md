# 프로젝트 개요
* 개발 기간 : 2025.11.03 ~ 2025.11.05
* 개발 인원 : 1인

# Tech Stack
* Language : Python
* UI Framework : CustomTkinter
* Library
    * pytubefix : 최신 유튜브 알고리즘 대응 및 스트림 추출
    * threading : 다운로드 및 병합 중 UI 프리징 방지
    * Tool
        * FFmpeg : 영상과 음성 스트림의 무손실 병합

# 핵심 기능
* 고화질 영상 소스 추출
    * Adaptive Stream 지원 : 유튜브에서 제공하는 최고 해상도의 영상 스트림과 고품질 오디오 스트림을 개별적으로 분석하여 가져옵니다
    * 스마트 파일 네이밍 : 제목 내 금지 문자(\/:*?"<>|)를 자동 제거하여 파일 시스템 오류를 방지합니다

* 실시간 다운로드 대시보드
    * 이중 진행률 표시 : 영상과 오디오의 다운로드 상태를 독립적인 프로그레스 바(0% ~ 100%)를 통해 실시간으로 시각화합니다
    * 타이머 기능 : 작업 시작부터 완료까지 소요되는 시간을 초 단위로 계산하여 사용자에게 제공합니다

* 자동 FFmpeg 병합 시스템
    * 심리스(Seamless) 병합 : 개별 다운로드된 영상과 음성을 FFmpeg를 사용하여 하나의 .mp4 파일로 자동 병합합니다
    * 자동 리소스 관리 : 병합이 성공적으로 완료되면 임시로 생성된 개별 소스 파일들을 자동 삭제하여 저장 공간을 효율적으로 관리합니다
    * 병합 상태 애니메이션 : 병합 중에는 '병합 중...' 애니메이션을 노출하여 프로그램이 정상 작동 중임을 사용자에게 알립니다

* 사용자 편의 기능
    * 최근 경로 기억 : config.json을 통해 사용자가 마지막으로 선택한 다운로드 폴더를 기억하여 재실행 시 자동으로 설정합니다
    * 원클릭 폴더 선택 : 시스템 폴더 브라우저를 연동하여 직관적인 경로 설정이 가능합니다
